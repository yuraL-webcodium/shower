Wordpress seed

You can find the seed here - Wordpress-seed at Bitbucket

For using this seed you should:

  1. Download latest wordpress
  2. Go to folder themes wp-content/themes.
  3. Here type and run
    git clone git@bitbucket.org:webcodium/wordpress-seed.git
  4. Type and run php wordpress-seed/set.php (Then at first you should type a current name theme then a new name of theme)

    You have to add a PATH environment for php in Windows OS.

The latest command does next:

  1. Removes and deregister no need files
  2. Rename theme
  3. Add cache busting
  4. Make new assets structure for project
  5. Sets up gulp
  6. Installs bower and npm dependncies
  7. Adds .gitignore
  8. Creates your own functions.php which will include to functions.php (name will be called functions-[new name of theme].php)
  9. Adds vendor.json and vendor.base.json

Developing:

  • For watching a project you should run gulp.
  • For building project for production you should run gulp --production.
  • In the functions-[new name of theme].php exists get_asset_version which allows load assets with cache busting
  • In the functions-[new name of theme].php can find commented string with called function wp_enqueue_style you can uncomment if you want load a Cuprum font as in example or change to other font if you need.
  • vendor.base.json - this file allows include js files which will be added to main js file (this seed includes jquery)
  • vendor.json - this file will copy files and will add to assets/vendor folder (this seed includes bootstrap)
